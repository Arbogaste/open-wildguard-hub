# M6 — Field Collection & Protected Human Intelligence (SCAFFOLD)

> Mission: structural channels for patrol data, citizen reporting, informant intel and opportunistic
> observations — each with its own anonymity, risk and verification level. Offline-first.

| Repo / tool | Take this |

| SMART Mobile / CyberTracker | fully offline patrol data collection + opportunistic sync |

| OpenRefine | dedupe + normalize amateur reports before they hit the DB |
| [DSA Alert Engine](https://github.com/Ruthwik9590/Tracking-and-Poaching-Alert-System-using-Advanced-Data-Structures) | offline CSV sync pattern |

## Note

quality — community programs fail when they measure ecology well but socio-economics poorly.
