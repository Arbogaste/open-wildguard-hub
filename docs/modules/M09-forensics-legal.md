# M9 — Evidence Chain & Legal Compliance (SCAFFOLD; chargesheet demo already in index.html)

> Mission: every event produces a court-ready, digitally-native evidence package — synced timestamp,
> coordinates, original media, hashes, sensor logs, operators, transformations, decisions. Continuous
> chain of custody so cases aren't thrown out.
